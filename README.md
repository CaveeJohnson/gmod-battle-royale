# Battle Royale

H1Z1-style version of Battle Royale. This is still in early alpha so expect alot of bugs.

## Installation
* Drag this folder into ```garrysmod/gamemodes```.
* Edit ```garrysmod/gamemodes/gmod-battle-royale/gamemode/sh_config.lua```.
